        \item[c)] $F(3) = \frac{k}{3} = 200 \Rightarrow k = 600$. So, $F(d) = \frac{600}{d}$.
        \item[d)] If the distance is doubled, force needed is halved. $F(2d) = \frac{600}{2d} = 300$N.
